## 📖 Overview
This repository contains the source code for the **AI Resume Maker** application, which was prototyped and live-coded ("vibe coded") during a Full Stack Development & AI Integration workshop held at **Tecnia Institute of Advanced Studies (TIAS)**.

The workshop served as a rapid accelerator, taking participants from full stack fundamentals to a working, AI-powered product in a single intensive session.

## 🛠️ What We Achieved

### Phase 1: Full Stack Crash Course
We kicked off the session by covering the foundational components of a modern web application:
1.  **Frontend (React):** Building component-based UIs and managing state.
2.  **Backend (Node.js/Express):** Understanding API endpoint creation.
3.  **Connectivity:** Connecting the React frontend to the custom backend server.

### Phase 2: Live-Coding "The Vibe" (Resume Maker App)
We built a real working product in real-time. The goal was to demonstrate how rapidly an idea can become a deployable MVP when using AI-assisted development tools (Antigravity, Cursor).

**Key features of the application include:**
*   Dynamic input forms for user data.
*   Integrating Large Language Models (LLMs) to automatically rewrite and optimize resume descriptions for specific job descriptions.
*   Connecting the frontend, backend, and AI layer into a single, cohesive user experience.

## 💻 Technical Prerequisities & Setup

If you wish to clone this repository and run the project locally, you will need:
*   [Node.js](https://nodejs.org/) installed (v18+ recommended)

### Installation

1.  **Clone the Repo:**
    ```bash
    git clone https://github.com/piyushnanwani/resume-builder.git
    cd resume-builder
    ```

2.  **Setup Frontend:**
    ```bash
    npm install
    npm run dev
    ```
